# MongoDB - Consultas con filtros

## Igualdad

```javascript
db.usuarios.find({
    edad: 25
})
```

o explícitamente:

```javascript
db.usuarios.find({
    edad: {
        $eq: 25
    }
})
```

---

## Mayor que

```javascript
db.productos.find({
    precio: {
        $gt: 100
    }
})
```

---

## Menor que

```javascript
db.productos.find({
    precio: {
        $lt: 100
    }
})
```

---

## Mayor o igual

```javascript
db.usuarios.find({
    edad: {
        $gte: 18
    }
})
```

---

## Menor o igual

```javascript
db.usuarios.find({
    edad: {
        $lte: 65
    }
})
```

---

## Distinto de

```javascript
db.usuarios.find({
    ciudad: {
        $ne: "Buenos Aires"
    }
})
```

---

## Dentro de una lista (`$in`)

```javascript
db.usuarios.find({
    ciudad: {
        $in: ["Buenos Aires", "Córdoba", "Rosario"]
    }
})
```

---

## No pertenece a una lista (`$nin`)

```javascript
db.usuarios.find({
    ciudad: {
        $nin: ["Buenos Aires", "Córdoba"]
    }
})
```

---

# Operadores lógicos

## `$and`

```javascript
db.usuarios.find({
    $and: [
        {
            edad: {
                $gt: 20
            }
        },
        {
            ciudad: "Madrid"
        }
    ]
})
```

Forma simplificada:

```javascript
db.usuarios.find({
    edad: {
        $gt: 20
    },
    ciudad: "Madrid"
})
```

---

## `$or`

```javascript
db.usuarios.find({
    $or: [
        {
            ciudad: "Madrid"
        },
        {
            ciudad: "Barcelona"
        }
    ]
})
```

---

## Combinar operadores

```javascript
db.usuarios.find({
    edad: {
        $gt: 20,
        $lt: 30
    },
    $or: [
        {
            ciudad: "Barcelona"
        },
        {
            ciudad: "Valencia"
        }
    ]
})
```

---

# Paginación

## Limitar resultados

```javascript
db.productos.find().limit(10)
```

---

## Omitir resultados

```javascript
db.productos.find().skip(10)
```

---

## Página 2 (10 por página)

```javascript
db.productos.find().skip(10).limit(10)
```

---

## Página 3 (5 por página)

```javascript
db.productos.find().skip(10).limit(5)
```

Fórmula:

```
skip = (pagina - 1) * cantidadPorPagina
```

---

# Mostrar resultados legibles

```javascript
db.usuarios.find().pretty()
```

---

# Ordenar resultados

Ascendente:

```javascript
db.usuarios.find().sort({
    edad: 1
})
```

Descendente:

```javascript
db.usuarios.find().sort({
    edad: -1
})
```

---

# Índices

Crear índice:

```javascript
db.usuarios.createIndex({
    edad: 1
})
```

Eliminar índice:

```javascript
db.usuarios.dropIndex({
    edad: 1
})
```

Ver índices:

```javascript
db.usuarios.getIndexes()
```

---

# Operadores de comparación

| Operador | Significado |
|----------|-------------|
| `$eq` | Igual |
| `$gt` | Mayor que |
| `$gte` | Mayor o igual |
| `$lt` | Menor que |
| `$lte` | Menor o igual |
| `$ne` | Distinto |
| `$in` | Dentro de una lista |
| `$nin` | No pertenece a una lista |

---

# Operadores lógicos

| Operador | Significado |
|----------|-------------|
| `$and` | Todas las condiciones |
| `$or` | Al menos una condición |
| `$not` | Niega una condición |
| `$nor` | Ninguna condición |
