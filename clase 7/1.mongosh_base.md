# MongoDB con mongosh - Comandos básicos

## 1. Instalar MongoDB Community Server

Descargar e instalar desde:

https://www.mongodb.com/try/download/community

Durante la instalación dejar seleccionada la opción de instalar **MongoDB Compass**.

---

## 2. Instalar mongosh

https://www.mongodb.com/try/download/shell

Para Windows descargar el paquete msi

## 2. Verificar la instalación

Abrir una terminal y ejecutar:

```bash
mongosh
```

Si aparece algo similar a:

```
Current Mongosh Log ID...
Connecting to: mongodb://127.0.0.1:27017/
```

la instalación fue correcta.

Salir de la consola:

```javascript
exit
```

---

# Bases de datos

## Listar todas las bases de datos

```javascript
show dbs
```

> Solo aparecen las bases de datos que contienen al menos una colección con datos.

---

## Crear o seleccionar una base de datos

```javascript
use colegio
```

Si no existe, **MongoDB no la crea inmediatamente**.

La base se crea cuando se inserta el primer documento.

---

## Ver la base de datos actual

```javascript
db
```

---

# Colecciones

## Listar colecciones

```javascript
show collections
```

o

```javascript
db.getCollectionNames()
```

---

## Crear una colección explícitamente

```javascript
db.createCollection("alumnos")
```

---

## Crear una colección implícitamente

No hace falta crearla previamente.

Al insertar el primer documento:

```javascript
db.alumnos.insertOne({
    nombre: "Juan",
    edad: 20
})
```

MongoDB crea automáticamente la colección **alumnos**.

---

# Insertar documentos

## Insertar un documento

```javascript
db.alumnos.insertOne({
    nombre: "Juan",
    edad: 20,
    curso: "Backend I"
})
```

---

## Insertar varios documentos

```javascript
db.alumnos.insertMany([
    {
        nombre: "Ana",
        edad: 22
    },
    {
        nombre: "Pedro",
        edad: 19
    }
])
```

---

# Consultar documentos

## Mostrar todos los documentos

```javascript
db.alumnos.find()
```

---

## Mostrar todos los documentos de forma legible

```javascript
db.alumnos.find().pretty()
```

> En versiones recientes de **mongosh**, `pretty()` ya no es necesario porque la salida es legible por defecto, pero sigue siendo útil conocerlo.

---

## Mostrar el primer documento

```javascript
db.alumnos.findOne()
```

---

## Buscar por un campo

db.coleccion.find(filtro)

```javascript
db.alumnos.find({
    nombre: "Juan"
})
```

---

## Mostrar solo algunos campos

```javascript
db.alumnos.find(
    {},
    {
        nombre: 1,
        edad: 1,
        _id: 0
    }
)
```

---

## Establecer límite

```
db.usuarios.find().limit(5)
```

---

## Proyección 

Trae todo de la base de datos (no filtra) pero solo muestra estos campos (si no quiero traer el id lo debo ocultar explicitamente):

```
db.usuarios.find({}, { nombre: 1, ciudad: 1, _id: 0 })
```

Tener en cuenta que no se puede mezclar inclusión (campos con :1) con exclusión (campos con :0) salvo el caso especial de _id

---

## Ordenar


```
db.usuarios.find().sort({ edad: -1 })
```

---

## Contar documentos

```javascript
db.alumnos.countDocuments()
```

---

# Eliminar documentos

## Eliminar un documento

```javascript
db.alumnos.deleteOne({
    nombre: "Juan"
})
```

---

## Eliminar varios documentos

```javascript
db.alumnos.deleteMany({
    curso: "Backend I"
})
```

---

# Actualizar documentos

## Actualizar un documento

```javascript
db.alumnos.updateOne(
    {
        nombre: "Juan"
    },
    {
        $set: {
            edad: 21
        }
    }
)
```

---

## Actualizar varios documentos

```javascript
db.alumnos.updateMany(
    {},
    {
        $set: {
            activo: true
        }
    }
)
```

---

# Eliminar una colección

```javascript
db.alumnos.drop()
```

---

# Eliminar la base de datos actual

```javascript
db.dropDatabase()
```

---

# Resumen rápido

| Acción                | Comando                               |
| --------------------- | ------------------------------------- |
| Entrar a mongosh      | `mongosh`                             |
| Salir                 | `exit`                                |

| Mostrar documentos    | `db.coleccion.find()`                 |
| Mostrar el primero    | `db.coleccion.findOne()`              |
| Buscar por filtro     | `db.coleccion.find({ campo: valor })` |
| Contar documentos     | `db.coleccion.countDocuments()`       |
| Actualizar            | `db.coleccion.updateOne()`            |
| Eliminar documento    | `db.coleccion.deleteOne()`            |
| Eliminar colección    | `db.coleccion.drop()`                 |
| Eliminar base         | `db.dropDatabase()`                   |
