# MongoDB - Actualización y eliminación

# Actualizar documentos

## Actualizar un documento

```javascript
db.usuarios.updateOne(
    { nombre: "Juan" },
    {
        $set: {
            edad: 25
        }
    }
)
```

---

## Actualizar varios documentos

```javascript
db.usuarios.updateMany(
    { ciudad: "Buenos Aires" },
    {
        $set: {
            activo: true
        }
    }
)
```

---

# Operadores de actualización

## `$set`

Crear o modificar un campo.

```javascript
db.usuarios.updateOne(
    { nombre: "Juan" },
    {
        $set: {
            ciudad: "Córdoba"
        }
    }
)
```

---

## `$inc`

Incrementar o decrementar un número.

Incrementar:

```javascript
db.usuarios.updateOne(
    { nombre: "Juan" },
    {
        $inc: {
            edad: 1
        }
    }
)
```

Decrementar:

```javascript
db.usuarios.updateOne(
    { nombre: "Juan" },
    {
        $inc: {
            edad: -1
        }
    }
)
```

---

## `$unset`

Eliminar un campo.

```javascript
db.usuarios.updateOne(
    { nombre: "Juan" },
    {
        $unset: {
            telefono: ""
        }
    }
)
```

---

# Combinar operadores

```javascript
db.usuarios.updateOne(
    { nombre: "Juan" },
    {
        $set: {
            ciudad: "Rosario"
        },
        $inc: {
            edad: 1
        }
    }
)
```

---

# Upsert

Actualizar si existe; insertar si no existe.

```javascript
db.productos.updateOne(
    {
        sku: "12345"
    },
    {
        $set: {
            precio: 100
        }
    },
    {
        upsert: true
    }
)
```

---

# Eliminar documentos

## Eliminar un documento

```javascript
db.usuarios.deleteOne({
    nombre: "Juan"
})
```

---

## Eliminar varios documentos

```javascript
db.usuarios.deleteMany({
    activo: false
})
```

---

## Eliminar todos los documentos

```javascript
db.usuarios.deleteMany({})
```

> Elimina todos los documentos, pero conserva la colección.

---

## Eliminar la colección

```javascript
db.usuarios.drop()
```

---

## Eliminar la base de datos

```javascript
db.dropDatabase()
```

---

# Buenas prácticas

- Verificar siempre el filtro antes de usar `updateMany()` o `deleteMany()`.
- Hacer respaldo antes de borrados masivos.
- Usar `updateOne()` cuando solo deba modificarse un documento.
- Utilizar `$inc` para contadores y stock.
- Usar `upsert` para sincronizar datos sin duplicados.

---

# Atomicidad

Las operaciones de actualización y eliminación son **atómicas a nivel de documento**.

Esto significa que un documento se actualiza completamente o no se actualiza en absoluto.

---

# Métodos principales

| Método | Descripción |
|---------|-------------|
| `updateOne()` | Actualiza el primer documento encontrado. |
| `updateMany()` | Actualiza todos los documentos encontrados. |
| `deleteOne()` | Elimina el primer documento encontrado. |
| `deleteMany()` | Elimina todos los documentos encontrados. |

---

# Operadores de actualización

| Operador | Función |
|----------|---------|
| `$set` | Crear o modificar un campo. |
| `$inc` | Incrementar o decrementar un número. |
| `$unset` | Eliminar un campo. |
| `upsert: true` | Insertar si el documento no existe. |
